# Sistema Integral Concretos Tepexi

Aplicación web (**Next.js**) en la carpeta [`web/`](./web/).

## Desarrollo local

```bash
cd web
npm install
npm run dev
```

## Variables de entorno

Ver `web/.env.example`. Configuración local en `web/.env.local` (no versionada).
